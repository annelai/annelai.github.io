Selected Projects

Constant Time Bilateral Filtering For Color Images

The bilateral filter is a commonly used technique in image processing. However, being nonlinear, it is computationally expensive. Several works have been proposed to accelerate the computation. Nevertheless, most techniques are tailored for gray-scale image or confined to specific kernel functions. We generalize the state-of-the-art constant time bilateral filtering technique for gray-scale images as a three-stage algorithm. Moreover, we proposed an extension for color images. Experimental results demonstrate the effectiveness of our solution.

Smart Projector

Painted Face Effect, where the textured contents are projected on the presenter’s face, frequently occurs during a speech. To remove this annoying effect, we implemented a real-time painted face removal algorithm on FPGA boards based on a projector-camera system. The system automatically correct the projecting contents without introducing any artifacts. Also, we developed a dynamic threshold decision module to function under light-variating environment.

Functionally Reduced And-Inverter Graph (FRAIG)

The FRAIG program can parse digital circuits described in the AIGER format and perform circuit optimization, including circuit size and simulation time. In this project, I designed my own data structure to identify functionally equivalent candidate (FEC) group and used Boolean Satisfiability (SAT) solver to prove the functional equivalence.

▫ Final project of #EE 3011 Data Structure and Programming

Automatic Speech Recognition (ASR) Model

The Automatic Speech Recognition (ASR) Model is able to recognize input speech feature, specifically MFCC and FBANK features of each time frame into word sequence. We implemented a phoneme (pronunciation units) classifier based on Deep Neural Networks (DNN) and used structured learning techniques to model the phoneme transition probabilities. Moreover, we implemented Recurrent Neural Network (RNN) based Language Model to improve speech recognition accuracy by re-scoring all possible sequences of N-best Viterbi algorithm.

▫ Final project of #CommE 5045 Machine Learning and Having It Deep and Structured
